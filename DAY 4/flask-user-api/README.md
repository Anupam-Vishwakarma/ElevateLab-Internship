# Flask User Management API

## 📌 Objective
Build a REST API using Flask to manage user data (CRUD operations).

## 🚀 Features
- Create a new user (POST)
- Get all users (GET)
- Get user by ID (GET)
- Update user (PUT)
- Delete user (DELETE)

## 🛠 Tools Used
- Python
- Flask
- Postman / Curl

## ⚙️ How to Run
```bash
git clone <your-repo-link>
cd flask-user-api
pip install -r requirements.txt
python app.py
